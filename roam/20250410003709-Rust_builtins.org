#+TITLE: Rust builtins

* Macros
** ~println!~ and ~print!~
   - Main standard output of the program.
   - Format input using ={}= and providing variable args.

** ~collection.iter()~ and ~collection.iter_mut()~
   - Provide iterators for collections.
** ~heapvar.clone()~
   - Perform a deep-copy and return the copy.
   - Expensive!
